# 🌱 Plant Tycoon

식물을 키우고 수확하여 부자가 되는 클리커 시뮬레이션 게임입니다!

## 🎮 게임 설명

**Plant Tycoon**는 일본의 인기 모바일 게임 "Nameko Cultivation(なめこ栽培)"에서 영감을 받아 제작된 웹 기반 방치형 게임입니다. 

### 게임 플레이
- 먹이를 주면 빈 공간에 식물이 자라기 시작합니다
- 식물이 다 자라면 클릭하여 수확할 수 있습니다
- 수확한 식물은 인벤토리에 쌓이며, 판매하여 골드를 획득할 수 있습니다
- 골드로 업그레이드를 구매하여 더욱 빠르게 성장시키세요!

### 희귀도 시스템
게임에는 5가지 희귀도의 식물이 있습니다:
- **Common (일반)** - 기본 버섯, 견과류, 곡물 등
- **Rare (레어)** - 허브, 베리류 등
- **Epic (에픽)** - 과일류
- **Legendary (전설)** - 꽃과 고급 과일
- **Mythic (신화)** - 다이아몬드, 왕관, 별 등 특별한 아이템

## ✨ 주요 기능

### 🏠 홈 화면
- 5x5 그리드 (총 25칸)에서 식물 재배
- 실시간 성장 시스템
- 클릭하여 수확 가능

### 🎒 인벤토리
- 수확한 아이템 자동 저장
- 동일한 아이템 자동 스택
- 개별 판매 (수량 선택 가능)
- 전체 판매 기능

### 🏪 상점
3가지 카테고리의 업그레이드를 구매할 수 있습니다:

**성장 속도 업그레이드**
- 비용: 100골드 + (레벨 × 50골드)
- 효과: 성장 속도 10% 증가 (레벨당)
- 최대 효과: 50% 빠른 성장

**희귀도 업그레이드**
- 비용: 1000골드 × 2^(레벨-1)
- 효과: 레어 이상 아이템 획득 확률 50% 증가 (레벨당)

**특수 기능 잠금 해제**
- 확률 통계 (2000골드): 각 희귀도별 획득 확률 표시
- 전체 수확 (2000골드): 모든 다 자란 식물을 한 번에 수확

### 📚 도감
- 발견한 모든 식물 기록
- 각 식물의 설명과 풍미 텍스트
- 누적 수확 횟수 추적
- 첫 발견 날짜 기록

### 🍖 먹이 시스템
4가지 종류의 먹이로 성장 속도를 조절할 수 있습니다:

| 먹이 종류 | 지속 시간 | 성장 속도 | 비용 |
|----------|----------|----------|------|
| 3분 무료 | 3분 | 0.5배 (느림) | 무료 |
| 1분 스피드 | 1분 | 2.0배 (빠름) | 200골드 |
| 5분 일반 | 5분 | 1.0배 (보통) | 150골드 |
| 10분 슬로우 | 10분 | 0.7배 (느림) | 350골드 |

- 먹이 효과는 언제든지 취소 가능
- 먹이를 주지 않으면 식물이 자라지 않습니다

## 🔧 기술 스택

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS
- **Storage**: LocalStorage (자동 저장)

## 🚀 실행 방법

### 개발 모드
```bash
cd plant-game-react
npm install
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.
