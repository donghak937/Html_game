# 🍄 버섯 재배 키우기 게임

귀여운 버섯을 키우고 수확하는 클리커 게임입니다!

## ✨ 주요 기능

- 🍄 **자동 재배**: 버섯이 자동으로 자라납니다
- 💰 **골드 시스템**: 버섯을 수확해서 골드를 모으세요
- ⚡ **업그레이드**: 골드로 재배 속도를 올릴 수 있습니다
- 📚 **도감 시스템**: 수확한 버섯을 도감에 기록하고 확인하세요
- 🎨 **희귀도 시스템**: 일반, 레어, 에픽, 레전더리 등급의 버섯
- 💾 **자동 저장**: 브라우저에 진행상황이 자동으로 저장됩니다

## 🚀 게임 실행 방법

**중요**: 이 게임은 JSON 파일을 로드하므로 로컬 웹 서버가 필요합니다.

### 방법 1: Node.js 사용 (추천)

1. Node.js가 설치되어 있다면:
```powershell
npx http-server -p 8000
```

2. 브라우저에서 접속:
```
http://localhost:8000/index.html
```

### 방법 2: Python 사용

1. Python 3가 설치되어 있다면:
```powershell
python -m http.server 8000
```

2. 브라우저에서 접속:
```
http://localhost:8000/index.html
```

### 방법 3: VS Code Live Server 확장 사용

1. VS Code에서 "Live Server" 확장 설치
2. `index.html` 파일에서 우클릭 → "Open with Live Server"


## 🎮 게임 방법

1. **버섯 수확**: 화면에 나타나는 버섯을 클릭하세요
2. **골드 획득**: 버섯마다 다른 가치의 골드를 얻을 수 있습니다
3. **속도 업그레이드**: 100골드로 재배 속도를 증가시킬 수 있습니다
4. **전부 수확**: 한 번에 모든 버섯을 수확합니다
5. **도감 확인**: "📚 도감 보기" 버튼으로 수집한 버섯을 확인하세요

## 📚 도감 기능

- **완성도 추적**: 발견한 버섯 수와 퍼센트 표시
- **필터링**: 희귀도별, 발견/미발견별로 필터링 가능
- **상세 정보**: 각 버섯의 골드 가치, 수확 횟수, 출현 확률 확인
- **NEW 배지**: 최근 24시간 이내 발견한 버섯 표시

## ⚙️ 설정 변경

버섯 종류나 확률을 변경하려면 `mushroom_types.json` 파일을 수정하세요:

```json
{
  "mushrooms": [
    {
      "emoji": "🍄",
      "value": 10,           // 골드 가치
      "rarity": "common",    // common, rare, epic, legendary
      "weight": 50,          // 출현 확률 (높을수록 자주 나옴)
      "description": "일반 버섯"
    }
  ],
  "settings": {
    "totalSlots": 24,              // 슬롯 개수
    "baseGrowthInterval": 3000,    // 재배 속도 (밀리초)
    "growthProbability": 0.3       // 버섯 생성 확률 (0.0-1.0)
  }
}
```
