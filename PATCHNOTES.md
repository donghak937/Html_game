# 📋 Plant Tycoon 패치노트

## Version 1.2.0 (2025-11-25)

### � UI/UX 개선
- **인벤토리 레이아웃 최적화**
  - 2열 고정 그리드 레이아웃 적용 (`repeat(2, minmax(0, 1fr))`)
  - 스크롤바 숨김 처리 (스크롤 기능은 유지)
  - 최대 높이 600px로 제한
  - **슬라이더 overflow 수정**: `flexShrink: 0`, `minWidth: 0`, `maxWidth: 100%` 적용
- **페이지 전환 UX**
  - 페이지 이동 시 자동 스크롤 최상단 이동  
- **상점 변경**
  - 성장 물약을 상점에서 제거 (기존 보유분은 인벤토리에 유지)

### 🐛 버그 수정
- 인벤토리 카드에서 슬라이더가 카드 경계 밖으로 삐져나오던 문제 해결

## Version 1.0.0 (2025-11-24)

### 🎉 초기 출시

#### 핵심 게임플레이
- 5x5 그리드 (25칸) 식물 재배 시스템
- 먹이를 주면 식물이 자라는 기본 메커니즘
- 클릭하여 수확하는 인터랙션

#### 🌱 식물 시스템
- **35종의 다양한 식물** 추가
  - Common (일반): 5종
  - Rare (레어): 9종
  - Epic (에픽): 10종
  - Legendary (전설): 10종
  - Mythic (신화): 5종
- 가중치 기반 랜덤 생성 시스템
- 각 식물마다 고유한 설명과 풍미 텍스트
- 희귀도별 차별화된 가치

#### 📦 인벤토리 시스템
- 수확한 아이템 자동 저장
- **동일 아이템 자동 스택** 기능
- **수량 선택 판매** 기능
  - 슬라이더로 판매할 수량 선택
  - 예상 획득 골드 실시간 표시
  - 확인/취소 버튼
- **전체 판매** 버튼
- 아이템별 판매 가격 및 개수 표시

#### 🏪 상점 시스템
- **4가지 업그레이드 카테고리**:
  - 성장 속도 업그레이드 (Lv.1~5)
  - 희귀도 확률 증가 (Lv.1~5)
  - 비료 업그레이드 (Lv.1~5)
  - 특수 기능 해금
- **소모품**:
  - 씨앗 폭탄 (빈 땅 전부 채우기)
  - 성장 물약 (모든 식물 즉시 성장)

#### 📚 도감 시스템
- 발견한 식물 자동 등록
- 희귀도별 필터링
- 총 수집률 표시
- 상세 정보 확인 (이름, 설명, 희귀도, 가치)

#### 🍖 먹이 시스템
- **3가지 먹이 옵션**:
  - 기본 먹이 (무료)
  - 프리미엄 먹이 (2000골드, 2배속)
  - 특급 먹이 (5000골드, 5배속)
- 성장 속도 가속 및 스폰 확률 증가
- 먹이 활성화 상태 표시
- 먹이 취소 기능 (환불 없음)

#### ⚡ 특수 기능
- **전체 수확** (1000골드)
  - 성장 완료된 모든 식물을 한 번에 수확
- **통계 보기** (500골드)
  - 현재 스폰 확률
  - 희귀도별 확률 분포
  - 성장 시간 정보

#### 🎨 UI/UX
- 모던하고 깔끔한 디자인
- 부드러운 애니메이션 (Framer Motion)
- 직관적인 탭 네비게이션
- 반응형 카드 레이아웃
- 시각적 피드백 (호버 효과, 색상 변화)

#### � 데이터 관리
- localStorage를 통한 자동 저장
- 새로고침 후에도 게임 진행 상태 유지
- 실시간 상태 업데이트

#### 🎯 게임 밸런스
- **Pity 시스템**: 스폰 실패 시 다음 확률 증가
- **가중치 시스템**: 희귀도별 적절한 출현 확률
- **경제 시스템**: 골드 획득 및 소비의 균형
- **진행 곡선**: 단계적 업그레이드를 통한 자연스러운 성장

#### 🚀 기술 스택
- React + Hooks
- Framer Motion (애니메이션)
- localStorage (데이터 저장)
- Vite (빌드 도구)
- GitHub Pages (배포)

- 웹 브라우저에서 플레이 가능
