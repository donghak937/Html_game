# 📋 Plant Tycoon 패치노트

## Version 1.0.0 (2025-11-24)

### 🎉 초기 출시

#### 핵심 게임플레이
- 5x5 그리드 (25칸) 식물 재배 시스템
- 먹이를 주면 식물이 자라는 기본 메커니즘
- 클릭하여 수확하는 인터랙션

#### 🌱 식물 시스템
- **35종의 다양한 식물** 추가
  - Common (일반): 5종
  - Rare (레어): 9종
  - Epic (에픽): 10종
  - Legendary (전설): 10종
  - Mythic (신화): 5종
- 가중치 기반 랜덤 생성 시스템
- 각 식물마다 고유한 설명과 풍미 텍스트
- 희귀도별 차별화된 가치

#### 📦 인벤토리 시스템
- 수확한 아이템 자동 저장
- **동일 아이템 자동 스택** 기능
- **수량 선택 판매** 기능
  - 슬라이더로 판매할 수량 선택
  - 예상 획득 골드 실시간 표시
  - 확인/취소 버튼
- **전체 판매** 버튼
  - 인벤토리의 모든 아이템을 한 번에 판매
  - 총 획득 골드 표시

#### 🏪 상점 시스템
**성장 속도 업그레이드**
- 비용: 100골드 + (레벨 × 50골드)
- 효과: 성장 속도 10% 증가 (레벨당)
- 최대 효과: 50% 빠른 성장
- ⚠️ **알려진 이슈**: 현재 실제로 적용되지 않음 (추후 수정 예정)

**희귀도 업그레이드**
- 비용: 1000골드 × 2^(레벨-1)
- 효과: 레어 이상 아이템 가중치 50% 증가 (레벨당)
- 레벨이 오를수록 희귀한 식물 획득 확률 증가

**특수 기능 잠금 해제** (각 2000골드)
- **확률 통계**: 각 희귀도별 획득 확률 실시간 표시
- **전체 수확**: 모든 다 자란 식물을 한 번에 수확하는 버튼

#### 🍖 먹이 시스템
4가지 종류의 먹이로 성장 속도 조절:

| 먹이 종류 | 지속 시간 | 성장 배율 | 비용 |
|----------|----------|----------|------|
| 3분 무료 | 3분 | 0.5배 (느림) | 무료 |
| 1분 스피드 | 1분 | 2.0배 (빠름) | 200골드 |
| 5분 일반 | 5분 | 1.0배 (보통) | 150골드 |
| 10분 슬로우 | 10분 | 0.7배 (느림) | 350골드 |

- 먹이 효과 남은 시간 실시간 표시
- **먹이 취소 기능**: 활성화된 먹이를 언제든지 취소 가능
- ⚠️ 먹이를 주지 않으면 식물이 자라지 않음

#### 📚 도감 시스템
- 발견한 식물 자동 기록
- **희귀도별 필터링** (전체/Common/Rare/Epic/Legendary/Mythic)
- 각 식물별 상세 정보
  - 첫 발견 날짜
  - 누적 수확 횟수
  - 식물 설명
  - 풍미 텍스트
- 미발견 식물은 물음표(❓)로 표시
- 발견률 표시 (예: 15/35 발견)

#### 💾 데이터 저장
LocalStorage를 통한 자동 저장:
- 골드
- 식물 상태 (위치, 성장 단계, 타입)
- 인벤토리
- 도감
- 업그레이드 레벨
- 잠금 해제 상태
- 먹이 상태 (활성화 여부, 남은 시간)

#### 🎨 UI/UX
- 4개 탭 네비게이션 (홈/인벤토리/상점/도감)
- 실시간 통계 표시 (골드/식물 수/아이템 수)
- 각 화면별 도움말 팁
- 부드러운 애니메이션
- 직관적인 아이콘 사용
- 한국어 인터페이스

#### 🐛 버그 수정
- 식물이 컨테이너 밖으로 나가는 시각적 버그 수정
- 버섯에서 식물로 모든 참조 이름 변경
- 인벤토리 아이템 스택 오류 수정
- 도감 데이터 중복 저장 문제 해결

#### 🔧 기술 스택
- React 18

## Version 1.1.0 (2025-11-25)

### ⚖️ 대규모 밸런스 및 시스템 개편

#### 🔄 시스템 변경
- **천장 시스템 (Pity System) 추가**
  - 식물 스폰 실패 시마다 확률이 점진적으로 증가합니다.
  - 운이 나빠도 일정 시간 내에 반드시 식물이 등장하도록 보장합니다.
- **성장 메커니즘 변경**
  - 이제 **비료(먹이)**가 있어야만 식물이 자랍니다.
  - 비료가 없으면 성장이 멈춥니다. 비료 관리가 중요해졌습니다!
- **비료 효과 변경**
  - 비료는 더 이상 성장 속도를 빠르게 하지 않습니다.
  - 대신 **스폰 확률**을 대폭 높여줍니다. (기본 1배 ~ 최대 2배)

#### 📉 밸런스 조정
- **기본 스폰 확률 감소**: 5% → **1%** (너무 흔한 스폰 방지)
- **성장 시간 조정**: 기본 8초 → **60초**
  - 업그레이드를 통해 성장 시간을 단축할 수 있습니다. (레벨당 -3초, 최소 5초)
- **업그레이드 비용 증가**: 후반부 밸런스를 위해 비용이 상승했습니다.
- **희귀 식물 등장 조건**:
  - **Epic**: 레어도 레벨 5 이상부터 등장
  - **Legendary**: 레어도 레벨 10 이상부터 등장
  - **Mythic**: 레어도 레벨 15 이상부터 등장

#### 🎨 UI/UX 개선
- **통계 기능 강화**
  - **현재 스폰 확률(%)** 실시간 표시 (Pity 스택 반영)
  - **성장 시간(초)** 실시간 표시
- **상점 가격 표기 오류 수정**: 실제 차감되는 골드와 표시되는 가격이 다르던 문제 해결
- **텍스트 가독성 개선**: 잘 보이지 않던 텍스트들을 검은색으로 변경하여 가독성 확보
